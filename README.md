 Функционал
- Добавление / редактирование / удаление технологий
- 3 статуса: Не начато → В процессе → Завершено (смена по кнопке)
- Сохранение всех данных в localStorage (после перезагрузки ничего не пропадает)
- Импорт и экспорт данных в JSON
- Красивый дашборд с прогресс-баром и статистикой
- Адаптивный дизайн (отлично выглядит на телефоне)
- Материальный дизайн (Material-UI v5)
- Навигация: Главная ↔ Детали технологии ↔ Дашборд

 Как запустить локально
```bash
# 1. Склонируй репозиторий
git clone https://github.com/ch-kost/kontrol4.git
cd kontrol4

# 2. Установи зависимости (один раз)
npm install

# 3. Запусти проект
npm start
 какие пакеты используется:
npm install @mui/material @mui/icons-material @emotion/react @emotion/styled react-router-dom
